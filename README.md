双向互相认真自签名证书例子

Netty SSL Server and OKHttps SSL client， 具体见代码。

script目录下有一些小工具

- setup-cert.sh // 构建证书
- export-cert // 导出证书到pem文件
- client.js // 一个node tls例子
- client_exportkeys.js // 使用导出key的node tls例子
